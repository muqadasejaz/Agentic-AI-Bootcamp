## 🚀 Overview

This repository is a structured, hands-on journey into **Agentic AI systems**, starting from LLM fundamentals and progressing toward building autonomous AI agents.

It focuses on understanding **how modern AI systems are designed, orchestrated, and deployed in real-world applications.**

---

## 🧠 What This Project Covers

This bootcamp explores the full stack of Agentic AI:

- Large Language Models (LLMs)
- Prompt Engineering
- Tool / Function Calling
- Retrieval-Augmented Generation (RAG)
- LangGraph Agent Workflows
- Multi-Agent Systems
- Real-world AI system design

---

## ⚙️ Why This Repository Exists

Most AI learning paths are:
- Too theoretical   
- Or too tool-based without understanding 

This project focuses on:
> Building deep intuition + practical system design skills for Agentic AI.
